## Getting Real Racing Car Models

### Free Sources:
1. **Sketchfab** (CC-BY license): https://sketchfab.com/search?q=racing+car&type=models
2. **TurboSquid Free**: https://www.turbosquid.com/Search/3D-Models/free/car
3. **Free3D**: https://free3d.com/3d-models/car
4. **Poly Haven**: https://polyhaven.com/models

### How to Add:
1. Download .glb or .gltf file
2. Place in `app/src/main/assets/models/cars/`
3. Update asset_config.json
4. Commit and push
